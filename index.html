질문을 직접 답하는 설문이 아니라, AI가 작성한 전체 응답을 붙여넣어 행동 벡터를 분석하는 도구입니다.

ChatGPT 열기 Gemini 열기 Claude 열기

사용자 흐름

Download -> Prompt -> Paste -> Analyze
1. 질문지 다운로드

버전이 포함된 질문 세트를 내려받아 AI에 그대로 입력합니다.

2. AI 응답 수집

AI가 작성한 전체 답변을 형식 그대로 복사합니다.

3. 응답 붙여넣기

제출 화면에 전체 텍스트를 넣고 분석을 실행합니다.

4. 행동 벡터 생성

파싱, 축별 분석, 요약 생성 후 Firestore에 저장합니다.

AI 응답 제출

AI의 전체 응답 텍스트를 붙여넣으세요. 형식은 Q1: 부터 Q60:, 그리고 마지막의 [END OF SURVEY]를 포함해야 합니다.

파싱 기준

Q번호: 헤더를 기준으로 질문별 답변을 분리합니다.

버전 관리

응답은 현재 질문 세트 버전과 함께 저장됩니다.

저장 구조

질문, 축, 제약, 분석 태그, 점수가 함께 Firestore에 기록됩니다.
